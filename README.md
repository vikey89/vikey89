### `> cat about.txt`

Android developer by day, indie maker by night. I craft mobile experiences with **Kotlin**, **Jetpack Compose**, and an unhealthy obsession with clean architecture. When I'm not shipping code, I'm running trails around Mount Etna or perfecting my pizza dough hydration ratios.

I'm obsessed with understanding **real user problems** before writing a single line of code. User-first thinking drives everything I build — if it doesn't solve a genuine pain point, it doesn't ship.
